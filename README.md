<!-- USAGE EXAMPLES -->
## Usage

To Run project run command  
   ```sh
   docker-compose up 
   ```
For migration part open second terminal and run command
   ```sh
   docker exec -it greenteam-web-1 bash 
   ```
After entering doker terminal run command 
   ```sh
    npm run typeorm:migration:generate -- init1
   ```
